Comunicazione periodica IVA / Gestisce la Comunicazione periodica IVA con l'elenco delle fatture emesse e

Generate xml file for sending to Agenzia delle Entrate, kwnown as Spesometro.

Gestisce la Comunicazione periodica IVA con l'elenco delle fatture emesse e ricevute e genera il file da inviare all'Agenzia delle Entrate. Questo obbligo è conosciuto anche come Spesometro light 2018 e sostistuisce i precedenti obbblighi chiamati Spesometro e Spesometro 2017.

Il softwware permette di operare in modalità 2017 per rigenerare eventuali file in formato 2017. Per eseguire questa funzione, prima di avviare Odoo eseguire la seguente istruzione:

::

export SPESOMETRO_VERSION=2.0
