Tax Authority Definitions/Definizioni Agenzia delle Entrate

This module has no any specific function for End-user. It contains various tax data for Italian account. Inside there are xml schema files used by FatturaPA, eInvoice and VAT settlement.

This module requires PyXB 1.2.4 http://pyxb.sourceforge.net/

Questo modulo non ha funzioni specifiche per l'utente finale. Contiene dati e definizioni per la fiscalità italiana. All'interno sono presenti gli schemi xml nei formati stabiliti dall'Agenzia delle Entrate usati da FatturaPA, fattura elettronica, Liquidazione IVA elettronica e Comunicazione IVA.

Attenzione! Questo modulo è incompatibile con i seguenti moduli OCA:

l10n_it_fatturapa versioni [7-11].0.2.0.0

Dallo schema di definizione dei file xml dell'Agenzia delle Entrate, pubblicato con urn:www.agenziaentrate.gov.it:specificheTecniche dipendono tutti i file xml da inviare all'AdE; se moduli diversi richiamano lo stesso schema dell'Agenzia delle Entrate nasce un conflitto che genera l'errore:

name CryptoBinary used for multiple values in typeBinding

Tutti i moduli che generano file xml destinati all'Agenzia delle Entrate devono dipendere da questo modulo. Questo è il motivo del conflitto con i moduli OCA.

Per maggiori informazioni visitare il sito www.odoo-italia.org o contattare l'autore.
